## Design Specifications
- Color Theme: Deep Wine Red, Amber Gold, and Soft Cream for a warm atmosphere.
- Typography: Serif layout for traditional Italian heritage styling.
- Responsiveness: Built using CSS Flexbox and Media Queries to support Desktop and Mobile screens.

## Local Setup
1. Clone the repository to your computer.
2. Navigate into the project folder.
3. Open the index.html file in any web browser.
